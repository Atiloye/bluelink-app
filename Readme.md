**BlueLink Social App**
================
##### (a linkedin clone)
----

**Description**
---------------

 This is a site made with inspiration from the Linkedin platform. It has a fully functional frontend and backend service courtesy of React, Vite, tailwind css, Daisy Ui, MongoDB, Mailtrap and Cloudinary.
